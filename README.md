# Middleman/DatoCMS example website

This repository contains a Middleman example integrated with a DatoCMS site. The site is deployed on Netlify, and can be seen at this URL: https://datocms-middleman-example.netlify.com/

To read more about the `middleman-dato` gem, please refer to [its documentation](https://github.com/datocms/middleman-dato).
